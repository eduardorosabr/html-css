Outras formatações

Código formatado e o tipo pre com code

Podemos usar a tag code para dizermos que se trata de um código

public static int soma(a,b) { int resultado = a+b; return resultado; }

Porém, se não usarmos a tag <pre>, o código não ira identar de forma correta, apenas usará uma fonte monoespaçada para melhor leitura do código

    public static int soma(a,b)
    {
        int resultado = a+b;
        return resultado;
    }
